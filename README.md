# Daisuke Akatsuka
[都立航空工業高等専門学校](https://ja.wikipedia.org/wiki/東京都立航空工業高等専門学校)在籍中から[四輪レーサー](https://www.driverdb.com/drivers/daisuke-akatsuka)を目指すも断念。[蕎麦屋](https://tabelog.com/tokyo/A1324/A132401/13092876/)、溶接屋を経てプログラマになる。[2003年IPA未踏ユース](https://www.ipa.go.jp/archive/jinzai/mitou/mitoipedia/2003youth-seika.html)（スーパークリエイタ）、[2006年IPA未踏ソフトウェア創造事業](https://www.ipa.go.jp/archive/jinzai/mitou/mitoipedia/2006kamiki-seika.html)採択。2011年[慶應義塾大学大学院政策・メディア研究科](https://www.sfc.keio.ac.jp)修了。現在は、[(株)Birchill](https://birchill.co.jp/)に所属し [Mozilla Firefox](https://www.mozilla.org/firefox/) の[Address Bar](https://support.mozilla.org/kb/search-firefox-address-bar)および[DevTools](https://firefox-dev.tools) を中心に開発を行う。2021年設立した [zeroinon](https://zeroinon.jp) では自分世代の高齢社会を考えている。
